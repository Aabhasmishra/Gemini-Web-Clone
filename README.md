## Setup

1. Get your Gemini API key from [Google AI Studio](https://ai.google.dev/)
2. Replace `Google_API_Key` in `script.js`
3. Open `index.html` in browser

```html
const Google_API_Key = "YOUR_API_KEY_HERE"; // Replace this
```
